# 网络特性

## 网络制式

1. 2G 不支持, 也不支持其语音功能
2. 3G 不支持, WCDMA/CDMA2000/TD-SCDMA均不支持
3. 4G 支持 Cat.1 模式, 不支持VoLTE, 不支持中国电信的短信服务
4. 5G 完全不支持

## TLS协议及加密套件

### TLS

1. TLS 1.0 支持
2. TLS 1.1 支持
3. TLS 1.2 支持
4. TLS 1.3 默认未开启, 未测试
5. SSL 3.0, 默认禁用
