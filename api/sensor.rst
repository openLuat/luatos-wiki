sensor.ds18b20(pin)
-------------------

获取DS18B20的温度数据

**参数**

+------------+--------------------------------------------------------+
| 传入值类型 | 解释                                                   |
+============+========================================================+
| int        | gpio端口号                                             |
+------------+--------------------------------------------------------+
| boolean    | 是否校验crc值,默认为true.                              |
|            | 不校验crc值能提高读取成功的概率,但可能会读取到错误的值 |
+------------+--------------------------------------------------------+

**返回值**

========== ==========================
返回值类型 解释
========== ==========================
int        温度数据,单位0.1摄氏度
boolean    成功返回true,否则返回false
========== ==========================

无
