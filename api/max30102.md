# max30102 - 心率模块(MAX30102)

## max30102.init(i2c_id,int)

初始化MAX30102传感器

**参数**

|传入值类型|解释|
|-|-|
|int|传感器所在的i2c总线id,默认为0|
|int|int引脚|

**返回值**

|返回值类型|解释|
|-|-|
|bool|成功返回true, 否则返回nil或者false|

**例子**

```lua
if max30102.init(0,25) then
    log.info("max30102", "init ok")
else
    log.info("max30102", "init fail")
end

```
